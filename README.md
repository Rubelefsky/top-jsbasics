# JavaScript Basics

A learning repository for [The Odin Project](https://www.theodinproject.com/paths/foundations/courses/foundations) JavaScript Basics course.

## About

This repository contains exercises and examples from The Odin Project's JavaScript Basics curriculum. It covers fundamental JavaScript concepts including variables, data types, operators, and console output.

## Topics Covered

- **Variables**: `let`, `const`, and understanding deprecated `var`
- **Variable reassignment**: Differences between `let` and `const`
- **Data types**: Numbers and strings
- **Basic operations**: Arithmetic operations and calculations
- **Console output**: Using `console.log()` for debugging and output

## Project Structure

```
.
├── basicscript.html    # Main HTML file that loads the JavaScript
├── javascript.js       # JavaScript examples and exercises
└── README.md          # This file
```

## How to Run

1. Clone this repository
2. Open `basicscript.html` in your web browser
3. Open the browser's Developer Console (F12 or Right-click → Inspect → Console)
4. View the output from the JavaScript examples

Alternatively, you can run the JavaScript file directly with Node.js:
```bash
node javascript.js
```
